An integrated precision agriculture platform that assists farmers through:

* Crop Recommendation
* Plant Disease Detection
* Fertilizer Advisory
* Weather Monitoring
* Agricultural Marketplace
* AI Chatbot Support
* Market Price Trend Analysis

## Tech Stack

* React + TypeScript
* Node.js + Express
* Firebase
* Gemini API
* Tailwind CSS

## System Architecture

The platform follows a three-layer architecture:

1. User Interface Layer
2. Processing Layer
3. Data & Knowledge Layer

## Research Motivation

Designed to support data-driven decision making in agriculture by combining multiple farming services into a unified platform.

## Getting Started

### Prerequisites

* Node.js
* Gemini API key
* Firebase project configuration

### Installation

```bash
npm install
```

### Environment Setup

Create a local environment file and add the required API keys and Firebase configuration values.

```bash
GEMINI_API_KEY=your_gemini_api_key
```

### Run Locally

```bash
npm run dev
```

## Key Modules

* AI-powered crop recommendation for precision farming decisions
* Plant disease detection support for early diagnosis
* Fertilizer recommendation based on agricultural inputs
* Weather-aware farming insights
